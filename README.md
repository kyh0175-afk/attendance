# Cosmos · 코스모스 독서시간 출석 관리 시스템

유봉여자고등학교 자율학습 프로그램 "코스모스 독서시간"의 출석 관리 웹 시스템입니다.

🔗 **배포**: [kyh0175-afk.github.io/attendance/v2/](https://kyh0175-afk.github.io/attendance/v2/)

---

## 스택

- **프론트엔드**: 단일 HTML (vanilla JS, 프레임워크 없음)
- **백엔드**: Supabase (PostgreSQL + Realtime)
- **호스팅**: GitHub Pages

---

## 주요 기능

### 교사용
- QR 기반 출석 체크 (4개 장소 동시 운영)
- 쌍둥이 장소 통합 (아우름 ↔ 교과1실)
- 실시간 출석 명단 조회
- 연장 안내 창 (20분 만료 → 10분 연장 1회)
- 수동 출석 / 사후 출석 추가
- 심야 감독 교사 자동 채움

### 학생용
- QR 스캔 → 학번 입력으로 출석
- 본인 출석 내역 조회 (주간/월간 뷰)
- 출석 수정 요청, 출석 요일 변경 요청

### 관리자용
- 학생 명단 관리 (엑셀 일괄 업로드)
- 출석 기록 조회·분석·다운로드
- 실시간 장소별 모니터링
- 진단 탭 (세션 상태, 교사 기기 로그)

### View 모드 (2026-04 추가)
비밀번호 없이 모든 교사가 출석 기록을 조회하고 사후 출석을 추가할 수 있는 읽기 전용 모드.
