# 코스모스 독서시간 출석 시스템

QR코드 기반 학생 출석 관리 시스템입니다.  
GitHub Pages + Google Apps Script + Google Sheets를 활용한 무료 서버리스 구성입니다.

---

## 🔗 접속 링크

| 구분 | URL | 비고 |
|------|-----|------|
| 메인 | `https://kyh0175-afk.github.io/attendance/` | 용도 선택 화면 |
| 출석용 (아우름) | `.../attendance/?teacher=아우름&from=attendance` | 북마크 권장 |
| 출석용 (교과1실) | `.../attendance/?teacher=교과1실&from=attendance` | 북마크 권장 |
| 출석용 (해오름) | `.../attendance/?teacher=해오름&from=attendance` | 북마크 권장 |
| 출석용 (리케이온) | `.../attendance/?teacher=리케이온&from=attendance` | 북마크 권장 |
| 확인용 | `.../attendance/?mode=view` | 비밀번호 입력 필요 |
| 관리용 | `.../attendance/?mode=admin` | 비밀번호 입력 필요 |

---

## 📋 주요 기능

### 출석용
- 장소별 QR코드 생성 및 수업 시작
- 학생 QR 스캔 → 학번 입력 → 출석 처리
- 실시간 출석 현황 (5초 폴링)
- 수동 출석 처리 (학번 직접 입력)
- 미등록 학생 현장 등록 (이름 + 출석 요일 입력)
- 사후 출석 처리 (수업 종료 후 추가)
- 수업 종료 시 미출석 학생 자동 결석 처리

### 확인용
- 출석 기록 조회 (프로그램 / 장소 / 학년 필터)
- 출석 분석 (주별 / 월별)
- 학생 명단 조회 (읽기 전용)

### 관리용
- 전체 탭 접근 가능
- 학생 명단 관리 (엑셀 업로드 / 직접 추가 / 삭제)
- 출석 요일 설정 (학생별 / 그룹별)
- 분석 시트 업데이트 (Google Sheets에 O/X 표 생성)

## 🔧 유지보수

### index.html 업데이트
1. `https://github.com/kyh0175-afk/attendance` 접속
2. `index.html` 클릭 → ✏️ 편집 → 전체 교체 → Commit changes

### Apps Script 재배포
1. Apps Script 편집기에서 코드 교체 후 저장
2. 배포 → 배포 관리 → ✏️ → 새 버전 선택 → 배포

### 트리거 설정 (최초 1회)
- 함수: `dailyUpdate` → 시간 기반 → 일 타이머 → 오전 0시~1시

## 🎓 학생 출석 방법

1. 선생님의 QR코드를 스마트폰 카메라로 스캔
2. 출석 페이지가 열리면 학번 5자리 입력
3. **출석하기** 버튼 클릭
4. 이름과 출석 시각이 표시되면 완료 ✅

> 처음 출석하는 학생은 이름과 출석 요일을 추가로 입력합니다.
